# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-19

### Added
- Curated library of 35+ micro-interactions and transitions for buttons.
- Card Spreads (Arc layouts, Linear spreads, Fans, Cascades).
- 3D Carousels (Carousel, Cover Flow, Time Machine).
- Amicro CLI for zero-configuration components add/install.
- Dynamic theme system supporting Light and Dark modes.
- Responsive sandbox layout with Grid, List, and Matrix view modes.
- Community standards documentation: `LICENSE`, `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md`, `SECURITY.md`.
- GitHub Issue templates (`bug_report.md`, `feature_request.md`) and Pull Request template.
- GitHub Actions CI workflow to verify builds automatically.

### Fixed
- Fixed mobile layout for filter buttons and grid options to fit narrow screens cleanly.
- Replaced traditional viewport height unit `100vh` with dynamic viewport height `100dvh` in main pages to resolve layout issues in mobile browsers.
