# Security Policy

## Supported Versions

We currently support and provide security updates for the following versions of Amicro:

| Version | Supported |
| ------- | --------- |
| < 1.0.x | Yes       |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please do **not** report it publicly through a GitHub issue. Instead, please report it securely:

1. Send an email to **Syed Subhan** at: [su2491251@gmail.com](mailto:su2491251@gmail.com)
2. Send a direct message on Twitter: [@SubhanHQ](https://x.com/SubhanHQ)
3. Or open a GitHub Security Advisory directly under this repository.

Please include as much detail as possible, including:
- A description of the vulnerability.
- Steps to reproduce or a proof of concept.
- The potential impact.

We will acknowledge receipt of your report within 48 hours and work with you to address and resolve the issue before public disclosure.
