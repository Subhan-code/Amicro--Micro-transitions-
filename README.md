## ✨ Features

- ⚡️ **Clean Micro-transitions**: Smooth, hardware-accelerated animations for buttons, controls, and UI states.
- ⚙️ **One-click Code Generation**: Copy-to-clipboard compiles pure React + Tailwind + Motion code instantly.
- 🌓 **Dynamic Theme System**: Seamless transitions between Dark and Light mode. Toggle theme copy matches.
- 📱 **Interactive layouts**: Live preview layouts in Grid, List, and Matrix modes with instant scaling animations.
- 📊 **Dynamic Stats**: Real-time integration of repository Stargazers counts.
- 🎨 **Harmonious Aesthetics**: Tailored styling inspired by Syed Subhan's **Oxygen UI**.

---

## 🛠️ Tech Stack

- **Framework**: React (with TypeScript)
- **Styling**: Tailwind CSS
- **Animations**: [Motion](https://motion.dev) (formerly Framer Motion v12)
- **Icons**: Lucide React
- **Build Tool**: Vite

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### 1. Clone the Repository
```bash
git clone https://github.com/Subhan-code/Amicro--Micro-transitions-.git
cd Amicro--Micro-transitions-
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```

## 📦 Adding Custom Micro-transitions

To add a new micro-transition to the sandbox list:
1. Define your transition's trigger properties in `src/data/buttons.tsx`.
2. Configure custom entry/exit behaviors in the `AnimatedButton` component wrapper in `src/components/AnimatedButton.tsx`.
3. Provide the copyable React source template in `src/utils/codeGenerator.ts`.
